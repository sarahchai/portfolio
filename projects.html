Projects

Code the Rant

This app is a social media type site with rant as a post and JavaScript code as comments. It is a full-stack CRUD application that the users can write, edit, and delete the post and comments. The users can post a rant about anything with anonymity. The users can code based on the rant and show off their cleverness with coding. Also, the users can like the post and comments and the number of likes is shown. I used a Code Mirror library as the code input box and also to render the JS code. On the profile page, the user can look for written posts and comments and liked posts and comments.

Remember Me

For this app, I worked with two of my classmates at General Assembly. This app is a full-stack CRUD application, which helps the user to store the new networking contact efficiently. The user can add, edit, and delete the contact and the contact list can be viewed by importance and by date added.

Art Gallery Exhibition Guide

This app is the list of exhibitions on Chelsea Art galleries that the user can filter the list by the date and location (street name). On the detail exhibition page, the hardcoded data of the gallery and exhibition information is on shown. Also, the artist information is given with Wikipedia API and the artist artworks and previous shows are shown with Artsy API.

Drop the Color

This game is finding a specific color on images. The color will be flashed for three seconds and the user needs to look for the color on movie stills. The user will select the color using an eyedropper on color input. After the selected color is submitted, the movie still with the answer is given to the user.
